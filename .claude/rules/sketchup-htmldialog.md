# UI::WebDialog and UI::HtmlDialog

Use `UI::HtmlDialog` instead of `UI::WebDialog` for creating UIs that require more control than what `UI.inputbox` and `UI.messagebox` provides. `UI::WebDialog` is legacy and should not be used in new code.

Reference examples: https://github.com/SketchUp/htmldialog-examples
